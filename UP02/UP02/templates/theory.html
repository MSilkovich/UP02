Ряды динамики. Аппроксимация функций. Метод наименьших квадратов.


Динамический ряд — это ряд однородных величин, который показывает изменение изучаемого фактора во времени.

Его можно рассматривать как статистическую форму для отображения развития параметра по времени.

Числа, составляющие динамический ряд, называются уровнями ряда. Они могут быть представлены абсолютными числами, относительными и средними величинами.


Средний уровень ряда рассчитывается как простое среднее арифметическое показателей динамического ряда:

$$\large{\overline{x} = {\sum x_i \over n}}$$


Для расчета абсолютного прироста, темпа роста и темпа прироста используют формулы:

Абсолютный прирост базовый:

$$\large{\Delta y_{бi} = y_i - y_0}$$

Абсолютный прирост цепной:

$$\large{\Delta y_{цi} = y_i - y_{i-1}}$$

Темп прироста:

Базовый: $$\large{T_{Пбi} = {\Delta y_{бi} \over y_0}}$$

Цепной: $$\large{T_{Пцi} = {\Delta y_{цi} \over y_{i-1}}}$$

Темп роста:

Базовый: $$\large{T_{рбi} = {\Delta y_{i} \over y_0}}$$

Цепной: $$\large{T_{рцi} = {\Delta y_{i} \over y_{i-1}}}$$

Средние значения этих показателей равны:

Средний абсолютный прирост: $$\large{\overline{\Delta y} = {y_n - y_0 \over {n - 1}}}$$

Средний темп роста: $$\large{\overline{T}_p = \sqrt[n]{T_1 * T_2 * T_3 * \ldots * T_n} }$$


Провести аналитическое выравнивание динамического ряда методом наименьших квадратов, т.е. функцию y = f(x) , заданную таблично, аппроксимировать многочленом первой степени y = P1(x) = a1 + a2x, для чего найдем коэффициенты a1 и a2 методом наименьших квадратов, решив систему уравнений:

Коэффициент а: $$\large{a = {\sum{x_i} \sum{y_i} - n \sum{x_iy_i} \over (\sum{x_i})^2 - n \sum{x_i^2}}}$$

Коэффициент b: $$\large{b = {\sum{x_i} \sum{x_iy_i} - \sum{x_i^2} \sum{y_i} \over (\sum{x_i})^2 - n \sum{x_i^2}}}$$

Коэффициент линейной парной корреляции: $$\large{r_{xy} = {n \sum{x_iy_i} - \sum{x_i} \sum{y_i} \over \sqrt{(n \sum{x_i^2} - (\sum{x_i})^2)(n \sum{y_i^2} - (\sum{y_i})^2)}}}$$

Коэффициент детерминации: $$\large{R^2 = r_{xy}^2}$$

Провести аналитическое выравнивание динамического ряда методом наименьших квадратов, т.е. функцию y = f(x) , заданную таблично, аппроксимировать многочленом второй степени y = P2(x) = a0 + a2x + a2x2, для чего найдем коэффициенты a1 и a2 методом наименьших квадратов, решив систему уравнений:

Система уравнений для нахождения коэффициентов a, b, c: $$\large{ \left\{ \begin{array}{l} a\sum{x_i^2} + b\sum{x_i} + nc = \sum{y_i}, \\ a\sum{x_i^3} + b\sum{x_i^2} + c\sum{x_i} = \sum{x_iy_i}, \\ a\sum{x_i^4} + b\sum{x_i^3} + c\sum{x_i^2} = \sum{x_i^2y_i}; \end{array} \right. }$$

Коэффициент корреляции: $$\large{R = \sqrt{1 - {\sum{(y_i - \overset{\frown}{y_i})^2} \over \sum{(y_i - \overline{y_i})^2}}},}$$

Где: $$\large{\overline y = {1 \over n} \sum{y_i}}$$

Коэффициент детерминации: $$\large{R^2}$$
